
## Navigation Tips

### Back and forth between Locations

| Command | Action |
|---------|--------|
| `ma`    | Marks the current cursor position with the mark 'a' |
| `` `a ``    | Jumps precisely to the line and column of mark 'a' [1] |
| `'a`    | Jumps to the start of the line where mark 'a' was set (useful for returning to a general area) [1] |
| `` `` ` `` | Jumps back to the exact previous location (like an undo for movement) [1] |
| `''`    | Jumps back to the start of the previous line [1] |
